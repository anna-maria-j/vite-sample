# Sample To Do List in Vite

- React with Typescript
- SCSS modules
- .env variables
- Tests in Jest and React Testing Library

## Quickstart

1. Clone this repository.
   ```
   git clone git@github.com:anna-maria-j/vite-sample.git
   ```

2. Install required dependencies
   ```
   npm install
   ```

3. Start project
   ```
   npm run dev
   ```
